Powerful Features

Everything you need for Persian development in VS Code

RTL Support

Professional right-to-left text support for Persian and Arabic languages with enhanced Copilot Chat experience.

Persian Calendar

Interactive Jalali calendar with date conversion tools and Persian month/day names.

Number Converter

Convert between Persian and English numbers, with number-to-words functionality in Farsi.

Arabic to Persian

Intelligent text conversion from Arabic characters to proper Persian equivalents.

JSON Parser

Advanced JSON parser with js-beautify integration, localStorage persistence, and interactive features.

IP & URL Tools

Comprehensive IP address and URL analysis tools with geolocation information.
